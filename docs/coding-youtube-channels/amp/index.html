Web Development Youtube Channels: The 11 Best You Must Watch

by Josh Ternyak

2020-05-07

Learning web development is a process of finding a great tutorial, watching or reading it, and then building projects on your own.

There are MANY youtube channels teaching web development. Here are the 11 best web development Youtube channels you must watch to improve as a web developer.

Let's get started.

Traversy Media

Whether you're looking for a web development crash course, or tutorials on different front-end frameworks and technologies, Brad Traversy has got you covered.

Not only does he have a Youtube channel with over 1,000,000 subscribers, he has courses on Udemy that will teach you everything you need to get hired at your dream job.

Programming with Mosh

Want to become a professional software engineer without a useless degree? Mosh says that all you need is passion and determination, and it's true. Universities make you pay for subjects and classes that you'll never actually use in the real world.

If you want to become a software developer right from you bedroom, then Programming With Mosh is a great Youtube channel to watch and learn from.

freeCodeCamp.org

freeCodeCamp is an open source community of busy people who learn to code and build projects for nonprofits. They have many in-depth and practical courses covering web development, computer science, backend, frontend, and everything in between.

If you want to become a professional software engineer or web developer, then freeCodeCamp.org is where it's at.

LearnCode.academy

LearnCode.academy is full of complete roadmaps, tutorials, and guides that will make you a skilled web developer and computer-wizard. They teach Javascript, Github, Docker, Jquery, SSH, Angular, CSS, HTML, machine learning, React, and more.

CS Dojo

CS Dojo makes videos mostly about programming and computer science. He talks about how to solve complex problems, web development essentials, Python, computer science, building apps with Django, Data Structures, Algorithms, and more.

Treehouse

Treehouse is an online school that teaches technology. They have courses on hundreds of topics including how to build a web app, website, mobile app, and how to start a business. If you're interested in becoming a web expert, then watching Treehouse's courses is your best bet.

Academind

Whether you want to learn Angular, Vue.js, or even Data Science, Academind has it all. They have in-depth courses on many web technologies. This includes CSS, Git & Github, React JS, Angular, Bootstrap, Flutter, and more. So, go watch their videos, and you'll be a better web developer afterwards.

Web Dev Simplified

Web Dev Simplified is all about teaching web development skills and techniques in a simple and practical way. If you're new to web development, this awesome channel has every tutorial and tool you need.

You'll learn the most popular and valuable technologies to convert you from a no stack to full stack web developer. Web Dev Simplified was made in order to teach and inspire new web developers.

If you're interested in learning CSS, Vanilla JS, JSON, and more, then this youtube channel is great to watch.

Dev Ed

Dev Ed teaches web development, web design, 3d modeling, and other popular technologies that will help you be as creative as you can be.

In a humorous way, he also teaches developers how to create video games in Unity or even develop an application in Node.js.

DesignCourse

If you're looking for courses on graphic design, frontend development, and even UI/UX design, Gary's your guy. He teaches in a fun and simple way that leaves you understanding what you just watched and listened to.

He focuses mainly on design, but also teaches some Javascript. Publishing new videos at 11:30 AM ET, you'll improve your web skills by listening to and watching his videos. He's been a course instructor and full stack dev for 20 years.

Andy Sterkowitz

Andy teaches about how web developers can maximize their productivity. He talks about ways to get better at getting better. Discussing his story of being a self taught programmer and how you could become a web dev expert as well.

If you're looking to maximize your time and learn how to be the best developer in the online world, then watching his videos is a great decision.

Conclusion

Now that you've got a list of the best web development Youtube channels to watch, now it's your turn. Spend time everyday learning from these 11 awesome web developers who put out valuable content, for free!
